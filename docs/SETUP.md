# Project Setup Guide

## Project Name

BioFuel Waste Tyre Pyrolysis Study

---

## Prerequisites

To open and review the project, the following tools may be needed:

- GitHub account
- Git
- Python 3.8 or higher
- Jupyter Notebook
- Simio
- Microsoft PowerPoint
- Web browser

---

## Clone the Repository

```bash
git clone <repository-url>
```

---

## Navigate to the Project Folder

```bash
cd BioFuel-Tala
```

---

## Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

## Open Project Documentation

The main documentation files are located inside the `docs/` folder.

Start with:

```text
docs/01_project_description.md
```

Then review the remaining documentation files in order.

---

## Open the Simulation Model

The Simio model file is located in:

```text
models/simio_model.spfx
```

This file requires Simio software to open.

---

## Open the Presentation

The PowerPoint presentation is located in:

```text
dashboards/BioFuel_Waste_Tyre_Pyrolysis.pptx
```

---

## Open the App Prototype

The BioFuel monitoring app prototype can be opened from this link:

[BioFuel Monitoring App](https://iofuel.oneapp.dev/)

---

## Repository Navigation

| Folder | Purpose |
|---|---|
| `docs/` | Project documentation |
| `data/raw/` | Original data and assumptions |
| `data/processed/` | Cleaned or transformed data |
| `notebooks/` | Python notebooks |
| `src/` | Source code and scripts |
| `dashboards/` | Presentation and dashboard materials |
| `models/` | Simulation model files |

---

## Notes

- The project documentation is written in Markdown.
- The Simio model requires Simio to run.
- The app prototype requires an internet connection.
- The project is a preliminary feasibility study and not a final industrial plant design.
