## Project Overview

This project evaluates the feasibility of a small-scale waste tyre pyrolysis system for BioFuel Company. The proposed system processes end-of-life tyres and converts them into useful outputs such as pyrolysis oil, carbon black, gas, light fuel fraction, and steel wires.

The project is designed as a Business Intelligence graduation project because it uses data, simulation outputs, operational indicators, financial calculations, environmental assumptions, and dashboard design to support decision-making. Instead of studying pyrolysis only as a chemical process, this project studies it as a complete production and business system.

The project combines four main analysis areas:

1. **Simulation analysis** using Simio.
2. **Economic feasibility analysis** using cost, revenue, profit, payback, and ROI calculations.
3. **Environmental assessment** based on waste reduction and resource recovery.
4. **Digital monitoring prototype** to support operational visibility and decision-making.

## Business Problem

End-of-life tyres are difficult to dispose of safely because they are non-biodegradable and can create environmental pressure when they are dumped, stored, or burned. At the same time, waste tyres contain recoverable materials and energy that can be converted into useful products through pyrolysis.

The main business problem is to determine whether a small-scale waste tyre pyrolysis system can operate efficiently, generate financial value, reduce environmental impact, and be monitored digitally to support better operational decisions.

---

## Research Question

The main research question is:

**Can a small-scale waste tyre pyrolysis system be technically feasible, economically profitable, environmentally beneficial, and digitally monitorable under the proposed BioFuel assumptions?**

---

## Project Objectives

The main objectives of the project are:

1. Develop a discrete-event simulation model for the waste tyre pyrolysis process.
2. Represent the full production flow from tyre receiving to final product separation.
3. Analyze throughput, waiting time, queue behavior, machine utilization, and reactor utilization.
4. Identify the main operational bottleneck in the system.
5. Estimate the capital investment, operating costs, revenues, profit, payback period, and ROI.
6. Evaluate the environmental benefits of converting waste tyres into useful outputs.
7. Design a digital monitoring application prototype for plant visibility.
8. Provide practical recommendations for improving system performance before real implementation.

---

## Project Scope

The project focuses on the preliminary feasibility evaluation of a small-scale waste tyre pyrolysis system. It includes:

- Process flow definition
- Simio simulation model
- Batching logic
- Capacity and utilization analysis
- Economic feasibility analysis
- Environmental impact discussion
- Digital monitoring app prototype
- Final results and recommendations

The project does not provide a final industrial engineering design. Real implementation would require supplier quotations, actual operating trials, environmental permits, product quality testing, and safety validation.

## Expected Value of the Project

The project helps decision-makers understand whether investing in a small-scale waste tyre pyrolysis plant is reasonable under the stated assumptions. It also highlights the reactor as the most critical system component, because improving reactor performance would improve production, revenue, and environmental outcomes at the same time.
